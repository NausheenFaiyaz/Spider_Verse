## 🕷️ SpiderVerse Explorer

Explore the Spider-Man multiverse like never before.

SpiderVerse Explorer is a React Native app that lets users discover different Spider-Man variants across multiple universes. With a cinematic intro, modern UI, and smooth navigation, the app delivers an immersive Spider-Verse experience.

## 🚀 Features
- 🎬 Cinematic splash screen (video intro)
- 🏠 Miles Morales inspired home screen
- 🔍 Search Spider-Man variants
- 🌌 Explore multiverse characters
- 📄 Detailed character information (powers, origin, abilities)
-  ❤️ Save favorite characters
- ⚡ Smooth navigation (Stack + Bottom Tabs)
- 🎨 Dark theme with red & blue SpiderVerse aesthetics

## 📱 Screens
- Splash Screen (Video Intro)
- Home Screen (Theme-based UI)
- Explore Screen (Search + Character List)
- Detail Screen (Full Character Info)
- Saved Screen (Favorites)

## 🛠️ Tech Stack
- React Native (Expo)
- TypeScript (TSX)
- React Navigation (Stack + Bottom Tabs)
- Expo Video (expo-video)
- REST API (Superhero API)
